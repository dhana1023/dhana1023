### About Me

I'm a Computer Science postgraduate (M.Tech, NIT Karnataka — CGPA 8.36/10) based in Auckland, New Zealand, working at the intersection of machine learning, data engineering, and real-world business problems. My thesis research focused on **user interest drift detection in real-time streaming recommendation systems**, using CUDA-accelerated inference, stratified reservoir sampling, and multi-view attention mechanisms.

Outside research, I've built production tools for retail forecasting, price anomaly detection, and real-time data pipelines, and I genuinely enjoy the point where a model stops being a notebook experiment and starts being something a business actually relies on.

- 🔬 **Research focus:** real-time recommendation systems, streaming data, time-series forecasting
- 🛠️ **Currently building:** production-style data pipelines and ML-driven analytics tools
- 📄 **Published in:** IEEE ISACC 2025, IEEE ICCCMLA 2022
